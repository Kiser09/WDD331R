Element Selectors

Element Selectors selects HTML elements based on the elements name. ex: p{ text-align:center; color: red; }

Universal Selector

The universal selector is designated with the * and selects all HTML elements on the page. ex: *{ text-align: center; color: blue; }

Grouping Selectors

The grouping selector selects all the HTML elements with the same style definitions. It a method of condensing CSS code when multiple elements share the same style defenitions. See the below example. original CSS ex: h1{ text-align: center; color: red } h2{ text-align: center; color: red } p{ text-align: center; color: red } grouping selector ex: h1, h2, p{ text-align: center; color: red }
